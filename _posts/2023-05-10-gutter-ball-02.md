### 🚀 airbnb 설정

{% highlight javscript linenos %}
npx install-peerdeps --dev eslint-config-airbnb
{% endhighlight %}

설치 후 `eslintrc.cjs` 파일의 `extends` 변경 (`eslint:recommended` 제거)

{% highlight javscript linenos %}
extends: [
  "airbnb",
  "airbnb/hooks",
  "plugin:react/recommended",
],
{% endhighlight %}

TypeScript 설정 추가

{% highlight javascript linenos %}
npm install -D eslint-config-airbnb-typescript
{% endhighlight %}

`@typescript-eslint/eslint-plugin`과 `@typescript-eslint/parser`는 install할 필요 없음

설치 후 `eslintrc.cjs` 파일의 `extends`에 추가

{% highlight javscript linenos %}
extends: [
  "airbnb",
  "airbnb/hooks",
  "plugin:react/recommended",
  "plugin:@typescript-eslint/recommended",
],
{% endhighlight %}

`parserOptions`에 `tsconfig.json` 파일 위치 추가

{% highlight javscript linenos %}
parserOptions: {
  ecmaVersion: "latest",
  sourceType: "module",
  project: "./tsconfig.json",
},
{% endhighlight %}

`tsconfig.json` 파일의 `include`에 `eslintrc.cjs` 파일 추가

{% highlight javscript linenos %}
"include": [".eslintrc.cjs", "src"],   
{% endhighlight %}

## PWA 세팅

### 🚀 `index.html` 수정

{% highlight html linenos %}
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
    <script>
      if ('serviceWorker' in navigator) {
        navigator.serviceWorker
          .register('/sw.js')
          .then(registration => {
            console.log('Service worker registration succeeded:', registration)
          })
          .catch(err => {
            console.log('Service worker registration failed:', error)
          })
      } else {
        console.log('Service workers are not supported.')
      }
    </script>
  </body>
{% endhighlight %}

<br>

### 🚀 `sw.js`

`public` 폴더에 `sw.js` 파일 생성

{% highlight javascript linenos %}
self.addEventListener('install', () => {
  console.log('[Service Worker] installed')
})

self.addEventListener('activate', e => {
  console.log('[Service Worker] actived', e)
})

self.addEventListener('fetch', e => {
  console.log(`[Service Worker] fetched resource ${e.request.url}`)
})
{% endhighlight %}
